合同正文（AI 生成）

第一条 合同标的及规格

1.1 甲方同意向乙方购买，乙方同意向甲方出售本合同附件一所列明的货物（以下简称"货物"）。附件一为本合同不可分割的组成部分。

1.2 货物的名称、规格、数量、单价及总价详见本合同附件一《物资明细表》。

1.3 本合同项下货物总价款为人民币壹佰贰拾捌万元整（¥1,280,000.00），含13%增值税。

第二条 交货方式及期限

2.1 乙方应在本合同签订后30天内将货物运至甲方指定地点：项目施工现场（江苏省南京市化工园区）。

2.2 货物运输费用由乙方承担，运输过程中的风险由乙方承担。

2.3 乙方完成交货后应通知甲方验收，甲方应在收到货物后7个工作日内完成验收。

第三条 质量标准及验收

3.1 乙方提供的货物应符合国家标准（GB/T 1499.2-2018）及本合同附件中约定的技术规格要求。

3.2 货物到场后，甲方有权对货物的数量、规格、外观进行验收。如发现数量短缺、规格不符或外观损坏，甲方应在验收后3个工作日内书面通知乙方。

3.3 质量保证期为货物验收合格之日起12个月。质保期内出现质量问题，乙方应在收到通知后48小时内响应并提供解决方案。

第四条 付款方式

4.1 本合同采用分期付款方式，具体付款比例如下：

  • 预付款：合同总价的30%，即¥384,000.00，在合同签订后7个工作日内支付；
  • 到货款：合同总价的40%，即¥512,000.00，在货物到场验收后7个工作日内支付；
  • 验收款：合同总价的20%，即¥256,000.00，在最终验收合格后7个工作日内支付；
  • 质保金：合同总价的10%，即¥128,000.00，在质保期满后14个工作日内支付。
第五条 违约责任

5.1 乙方逾期交货的，每逾期一日，应按逾期交付货物总价的0.1%向甲方支付违约金。逾期超过30日的，甲方有权解除合同。

5.2 甲方逾期付款的，每逾期一日，应按逾期支付金额的0.1%向乙方支付违约金。

5.3 因货物质量不符合合同约定，乙方应承担免费更换、修理或退货的责任，并赔偿由此给甲方造成的直接经济损失。
