# 采购订单功能改进设计文档

## 概述

对采购订单（订购单）和请购单模块进行多项改进，包括订单编号自动生成、采购人引用当前用户、供应商数据显示修正、日期格式修正、明细数量可编辑、单位必填、移除 AI 功能、新增 Excel 导入、隐藏物料编码等。

## 涉及页面和文件

| 文件 | 改动说明 |
|------|---------|
| `src/lib/numbering.ts` | 新增 `generateOrderNo()` 函数 |
| `src/lib/validations.ts` | `requisitionItemSchema.unit` 改为必填 |
| `src/app/projects/[id]/procurement/orders/new/page.tsx` | 多项修改（见下方） |
| `src/app/projects/[id]/procurement/requisitions/new/page.tsx` | 移除 AI、新增 Excel 导入、单位必填 |
| `src/app/projects/[id]/procurement/requisitions/[reqId]/page.tsx` | 隐藏物料编码列、单位必填标记 |
| `src/app/projects/[id]/procurement/requisitions/[reqId]/edit/page.tsx` | 单位必填标记 |
| `prisma/seed-dev.ts` | 供应商数据修正（UUID → 实际名称） |

## 需求明细

### 1. 订单编号自动生成

- 新增 `generateOrderNo()`：格式 `PO-YYYYMMDD-NNN`（如 `PO-20260603-001`）
- 订购单新建页面加载时自动调用，显示为只读展示行
- 移除手动输入 `orderNo` 的文本框

### 2. 采购人引用当前用户

- 表单加载时 `purchaser` 自动填入 `user.name`
- 改为只读展示行（同请购单页面风格）

### 3. 供应商名称修正

- 数据库测试数据中 `Partner.name` 字段存的是 UUID 而非实际名称
- 清理测试数据，用户可重新添加供应商

### 4. 需求日期中文格式

- 订购单明细的 `requiredDate` 当前使用 `Date.toString()` 显示英文
- 改为格式化显示 `YYYY-MM-DD`

### 5. 订购单明细数量可编辑

- `quantity` 字段取消 `readOnly`，允许用户在订购单中直接修改

### 6. 请购单单位必填

- Schema：`requisitionItemSchema.unit` 从 `z.string().default('')` → `z.string().min(1, '请选择单位')`
- 请购单新建/编辑表单中单位标签加红色 `*` 标记

### 7. 移除 AI 功能

**订购单新建页面：**
- 移除 `AiFillButton` 组件引用和渲染

**请购单新建页面：**
- 移除"一句话快速建单"输入框和解析按钮
- 移除 `ExtractMaterialsDialog` 组件
- 移除相关图标引用（`Sparkles`, `Loader2`）和状态变量（`quickText`, `parsing`）

### 8. Excel 导入物料明细

- 替换请购单新建页面中原 AI 区域
- 支持 `.xlsx` / `.xls` 格式
- 使用 `xlsx` 库在客户端浏览器端解析（无需上传服务端）
- 列名映射：`物料名称`、`规格`、`材质`、`牌号`、`标准规范`、`数量`、`单位`、`用途`、`需求日期`
- 解析成功后自动填充到明细列表

### 9. 隐藏物料编码

物料编码为内部控制数据，从以下页面移除显示：
- 请购单详情页面表格
- 订购单新建页面的请购单明细列表
- 订购单新建页面的订购单明细列表

## Excel 导入解析逻辑

1. 用户点击"导入 Excel"按钮，选择文件
2. 使用 `xlsx` 库读取文件内容
3. 取第一个 sheet 的数据
4. 列名映射关系：

| Excel 列名 | 表单字段 | 类型 | 是否必填 |
|-----------|---------|------|---------|
| 物料名称 | materialName | string | 是 |
| 规格 | specification | string | 否 |
| 材质 | material | string | 否 |
| 牌号 | materialGrade | string | 否 |
| 标准规范 | applicableStandard | string | 否 |
| 数量 | quantity | number | 是 |
| 单位 | unit | string | 是 |
| 用途 | purpose | string | 否 |
| 需求日期 | requiredDate | date | 否 |

5. 解析成功后追加到表单明细列表，并提示成功导入条数

## 边界情况和注意事项

- **订单编号**：自增序号按日重置（同请购单编号规则）
- **Excel 导入**：空文件提示"文件中无有效数据"；列名不匹配时提示"请检查 Excel 列名格式"
- **单位必填**：导入时若单位列为空，会触发表单校验错误提示
- **数量编辑**：修改数量后单价不变，小计（总金额）需联动重新计算
