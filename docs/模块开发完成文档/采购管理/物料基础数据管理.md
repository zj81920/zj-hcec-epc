# 物料基础数据管理

## 需求概述
提供物料主数据管理功能，用户在创建请购单录入物料时自动将物料存入物料库，并支持物料名称搜索联想与自动编码，无需提前维护物料数据。

## 功能清单
1. **物料自动入库** - 首次录入的物料自动存入物料库，无需手动提前维护
2. **物料名称搜索联想填充** - 输入物料名称时自动搜索匹配的物料，选中后自动填充规格、材质、牌号、标准等字段
3. **物料编码自动生成** - 物料编码按 `MAT-XXXXXX` 格式自动生成（如 MAT-000001）
4. **搜索 API** - `GET /api/materials/search?q=xxx`，支持按物料名称模糊搜索

## 注意事项
- 物料库会随请购单录入自动增长，无需提前维护或批量导入
- 搜索联想 API 返回结果包含物料编码、名称、规格、材质、牌号、标准等完整字段
- 物料编码生成规则：MAT- 前缀 + 6 位数字序号，从 000001 开始递增
