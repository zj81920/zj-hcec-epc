# 采购订单管理

## 需求概述
从已批准的请购单中选择物资，生成采购订单，完成采购全流程管理。支持直接采购和询价采购两种模式，集成 AI 历史参考价辅助比价决策。

## 功能清单

### 采购订单流程与创建
1. **采购订单创建** — 从已批准且未关闭的请购单中选择物资，生成采购订单
2. **订单编号自动生成** — 格式为 PO-YYYYMMDD-NNN（如 PO-20260605-001），与请购单 REQ 格式对应
3. **订单头部信息** — 包括订单编号、采购人（当前用户自动填充，只读）、供应商（搜索式弹出面板选择）、联系人、电话、收货地址、交货日期
4. **从请购单选择物资** — 列表展示可勾选的请购单项，支持从多个请购单中挑选物资
5. **明细行管理** — 物料名称、规格、数量（可编辑）、单位、单价、小计，底部自动汇总合计金额
6. **合计金额实时计算** — 数量/单价变更时合计实时准确更新
7. **需求日期格式** — 统一为 YYYY-MM-DD 格式显示

### 双状态管理
8. **流程状态**（全局统一 4 值） — 草稿 → 审批中 → 已批准 → 已驳回
   - 控制编辑/删除权限：仅"草稿"状态可编辑和删除
   - 后续对接正式审批流程时，审批节点对应此状态
9. **业务状态** — 询价中 → 待发货 → 已发货 → 已完成
   - 询价中：询价采购审批通过后，供应商报价阶段
   - 待发货：订单生效，等待供应商发货
10. **合同标识** — 预留 `contractNo` 字段，待采购合同模块回填后作为防重复签合同标识

### 采购方式（询价采购模块）
11. **直接采购** — 选 1 家供应商 + 手动填价格，审批通过后业务状态进入待发货（当前模式）
12. **询价采购** — 选多家供应商 → 审批通过 → 供应商在线报价（多轮） → 报价对比选标 → 自动回填价格
13. **采购方式切换** — 草稿状态下可自由切换，切换时清空不兼容数据
14. **询价链接生成** — 为每家有 Token 的唯一链接，两步式验证（公司名称校验 + 报价表单）
15. **多轮报价** — 采购员可对单个供应商发起新一轮报价，供应商截止前可自由修改
16. **在线报价页** — 供应商免登录报价，显示项目名/物资明细/要求到货时间/采购方技术文件，支持上传技术文件
17. **报价对比（Tab 切换）** — 采购员可逐家查看报价明细，选中后价格自动回填到订单
18. **选标定标** — 选中一家供应商后，供应商信息 + 各明细单价自动回填，业务状态从询价中推进到待发货

### AI 历史参考价
19. **AI 参考价列** — 订单物资明细表右侧显示历史参考单价、参考供应商、参考品牌、参考日期、AI 判断标签
20. **三数据源优先级** — 当前项目订单 → 全局历史订单 → 历史数据表（兜底）
21. **AI 语义匹配** — 通过物料名称、规格、材质维度的语义相似度匹配，返回 confidence 等级
22. **价格合理性判断** — AI 返回高/中/低置信度标签，辅助比价决策

### 历史采购数据管理
23. **Excel 批量导入** — 支持按模板上传历史采购数据，程序解析直接入库
24. **列表明细管理** — 支持分页查询、关键词搜索、逐行编辑和删除
25. **模板下载** — 提供与采购单明细格式一致的下载模板

### 供应商与合作方管理
26. **供应商选择** — 搜索式弹出面板（Combobox），输入关键词实时过滤，从合作方管理中的"供应商"类型选取
27. **合作方管理**（公司管理下） — 管理供应商/分包商/服务商，包含基本信息、联系方式、财务信息、资质评级

### 数据校验与防护
28. **重复采购防护** — 已选择过的请购单项不可重复添加，防止同一物资被多次采购
29. **数量/单价非负校验** — 数量、单价输入值不可为负数，前端实时拦截
30. **请购单位必填** — 请购单位字段为必填

### 审批与校验
31. **审批校验** — 创建采购订单时，系统校验所选请购单流程状态必须为"已批准"且业务状态不为"关闭"
32. **删除校验** — 仅"草稿"状态的订单可删除，删除后回收已占用的采购数量

### Excel导入物料
33. **Excel导入物料明细** — 支持 .xlsx/.xls 格式文件导入，客户端直接解析

### 界面优化
34. **隐藏物料编码列** — 物料编码为内部数据，不在界面展示
35. **订单详情询价进度** — 询价中订单展示供应商横条卡片 + Tab 切换报价对比

## 注意事项
- **双状态设计**：流程状态管权限（草稿/审批中/已批准/已驳回），业务状态管进度（询价中/待发货/已发货/已完成），互不干扰
- 流程状态全局统一 4 值，已写入项目规则 `.trae/rules/project_rules.md`
- 创建采购订单后，对应请购单的**业务状态**会自动更新（正常→部分采购，全部采购→关闭）
- 删除采购订单时，系统会回收已占用的采购数量，恢复请购单的可采购额度
- 只有流程状态为"已批准"且业务状态不为"关闭"的请购单才能创建采购订单
- 供应商必须从合作方管理中类型为"供应商"的记录中选择
- 预留了 `contractNo` 字段，采购合同模块开发后回填该字段用于防止重复签合同
- Excel 导入由客户端直接解析，文件上传至服务器
- **询价 Token 安全**：Token 存储在 `OrderQuote.token`，唯一索引防碰撞，每个链接绑定特定供应商
- **供应商报价验证**：两步式 — 先输入公司全称精确匹配，通过后进入报价页
- **多轮报价数据隔离**：每轮数据按 `round` 字段隔离，不保留同轮内修改历史
- **选标回填**：选中后供应商信息和报价数据写入订单 `supplier*` 和 `OrderItem.unitPrice/totalAmount`
- **AI 参考价空数据处理**：历史数据全空时跳过 AI 调用直接返回 null，节省 Token
