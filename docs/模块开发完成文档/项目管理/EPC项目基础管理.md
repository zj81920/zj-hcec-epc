# EPC项目基础管理

## 需求概述
EPC工程项目全生命周期基础管理，提供项目创建、项目看板、导航布局及里程碑追踪等核心能力，是EPC管理系统的基座模块。

## 功能清单
1. **项目CRUD** - 支持新建项目、编辑项目信息、项目列表分页查询、项目详情查看，包含项目名称、类型、状态、合同金额、计划时间等字段
2. **多项目类型支持** - 支持 EPC / E+P+C / EPCM / PC / 设计咨询 等多种工程承包模式，不同类型在导航和功能菜单上有所区分
3. **项目仪表盘** - 项目概览看板，展示项目进度环（完成百分比）、里程碑时间轴（甘特图风格）、快捷入口（跳转到设计/施工/HSE等子模块）
4. **项目导航布局** - 顶部主导航（切换模块）+ 左侧子导航（切换功能页），基于 Next.js App Router 的布局嵌套实现
5. **里程碑管理** - 里程碑的创建、编辑、完成状态标记，支持按时间线展示，进度环根据里程碑完成度自动计算
6. **文件存储抽象层** - 封装统一的文件存储接口（`FileStorage`），当前实现为本地磁盘存储（`LocalFileStorage`），后续可切换至阿里云 OSS / MinIO 等，业务代码无需改动

## 注意事项
- 项目类型的枚举定义在 `ProjectType` 中，新增类型需要同步更新导航菜单的路由映射
- 进度环依赖里程碑完成率计算，里程碑数据变动后需触发进度重新计算
- 文件存储抽象层使用依赖注入，新增存储后端只需实现 `FileStorage` 接口并在 `di container` 中注册
- 导航布局采用 `(project)/layout.tsx` 嵌套结构，修改导航项需同时更新顶部导航和左侧子导航的配置
- 数据库使用 Prisma + PostgreSQL，项目表 `Project` 的字段变更需运行 `npx prisma migrate` 同步
