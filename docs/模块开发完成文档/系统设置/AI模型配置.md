# AI模型配置

## 需求概述
在公司管理 → 系统设置下，支持配置多个 AI 提供商，包括 API 地址、密钥、模型等参数，并支持测试连接验证可用性。

## 功能清单
1. **AI模型配置管理** — 位于公司管理 → 系统设置菜单下
2. **多提供商支持** — 支持 OpenAI、Anthropic、Azure、自定义（兼容 OpenAI 协议）四种提供商类型
3. **可配置参数** — API 地址（Base URL）、API Key（加密存储）、模型名称（如 gpt-4o、claude-3-opus 等）
4. **测试连接** — 一键测试连通性，验证配置是否可用
5. **配置管理** — 支持新增、编辑、删除、启用/禁用 AI 模型配置

## 注意事项
- API Key 在存储时经过加密处理（非明文存储），数据库层面不可逆，确保安全性
- 配置保存后建议立即进行"测试连接"确认可用，避免后续调用时发现不可用
- 自定义提供商需兼容 OpenAI 的 API 协议格式才能正常使用
- 不同提供商支持的模型名称不同，请参考各官方文档填写
