# 专业基础数据

## 需求概述
管理工程项目中的专业字典数据，在系统设置中维护，预设 12 个常用专业，供请购单、采购订单等模块引用。

## 功能清单
1. **专业字典管理** - 在公司管理 → 系统设置下提供专业数据维护页面
2. **预设专业** - 默认包含 12 个专业：土建、设备、工艺、电气、仪表、管道、暖通、给排水、总图运输、结构、通信、消防
3. **专业编码自动生成** - 根据中文名称自动转为拼音首字母作为编码（如"土建"→"TJ"）
4. **业务引用** - 请购单中可选择专业，采购订单从请购单自动继承专业信息
5. **增删改支持** - 支持新建、编辑、删除专业

## 注意事项
- 如果专业列表有调整，可以在系统设置中自行增删，无需修改代码
- 专业编码生成规则：取中文名称每个字的拼音首字母，大写；如遇多音字以常见读音为准
- 删除专业前需确认该专业未被任何请购单或采购订单引用
