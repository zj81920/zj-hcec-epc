# HSE管理模块

## 需求概述
管理EPC项目中的健康、安全与环境（HSE）相关记录，包括事故事件、安全检查和安全培训三大核心功能，支撑项目安全管理的闭环运作。

## 功能清单
1. **事故事件记录** - 记录项目现场发生的事故或未遂事件，包含以下字段：
   - 事件类型：人身伤害 / 火灾 / 泄漏 / 未遂事件 / 其他
   - 严重程度：一般 / 较大 / 重大
   - 发生时间、地点、涉事人员
   - 事件描述、原因分析、纠正与预防措施
   - 关闭状态（待处理 / 处理中 / 已关闭）
2. **安全检查记录** - 定期或专项安全检查的记录与跟踪：
   - 检查区域、检查日期、检查人员
   - 发现的问题清单（逐条记录，含问题描述、照片附件）
   - 整改要求与责任人、整改期限
   - 状态跟踪：待整改 → 整改中 → 已复查 → 关闭
3. **安全培训记录** - 记录安全教育培训开展情况：
   - 培训主题、培训日期、讲师、参训人数
   - 培训内容描述、培训对象
   - 支持上传培训课件 / 签到表 / 考试记录等附件
4. **附件上传功能** - 支持图片、文档等附件的上传和查看，安全检查问题照片支持多图上传，培训记录支持课件和签到表上传

## 注意事项
- 严重程度评估标准需与项目 HSE 管理手册保持一致，建议在枚举值上附加说明提示
- 事故事件一旦创建不可删除（审计合规要求），仅可关闭或归档
- 安全检查的问题列表使用 JSON 数组或子表存储，每条问题独立跟踪状态
- 安全培训记录中的"参训人数"建议做数据校验，不允许为负数或超过项目总人数
- 所有HSE记录建议增加审计日志（创建人、创建时间、最后修改人、最后修改时间），满足安全合规审计要求
- 附件上传同样使用文件存储抽象层
