# 设计管理模块

## 需求概述
管理EPC项目中的设计交付物与沟通记录，包括设计文件库、设计联络单和设计会审记录，确保设计过程可追溯、可管控。

## 功能清单
1. **设计文件库** - 支持上传和在线查看设计文件（PDF / DWG / DOCX），按专业（工艺/管道/结构/电气/仪表等）和分类（初步设计/详细设计/施工图等）进行管理，提供文件列表检索与筛选
2. **设计联络单** - 设计方与业主/施工方之间的正式沟通记录，支持发送联络单和对联络单进行回复；状态流转：`待回复` → `已回复` → `已关闭`，每条联络单记录发送方、接收方、标题、正文及附件
3. **设计会审记录** - 记录设计会审的日期、地点、参与人员、会审结论及待办事项，支持上传会审纪要附件
4. **文件版本管理** - 上传新版本时保留历史版本，用户可查看版本历史、下载任一版本；版本号采用语义化递增（v1.0 → v1.1 → v2.0）

## 注意事项
- DWG 文件在浏览器中无法直接预览，前端使用 `查看` 功能时需提示用户下载并通过本地 CAD 软件打开
- 文件版本管理：删除文件时仅标记删除状态而非物理删除，保留版本历史记录
- 设计联络单的状态变更需校验权限：`已关闭` 状态不可再回复，仅创建人或管理员可关闭
- 专业和分类建议使用枚举或字典表维护，避免硬编码，便于后续扩展
- 文件上传依赖文件存储抽象层，大文件（>50MB）需注意前端分片上传或超时处理
