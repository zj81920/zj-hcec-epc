# 施工管理模块

## 需求概述
管理EPC项目施工现场的任务分配与资料归档，涵盖施工任务追踪和施工资料分类存储，支撑现场管理的规范化运作。

## 功能清单
1. **施工任务管理** - 任务的创建、编辑、删除与列表展示，字段包括：任务名称、施工区域（分区/标段）、计划开始/结束时间、实际进度（百分比）、负责承包商、负责人、任务状态（待开始/进行中/已完成/延期）
2. **施工资料管理** - 按资料分类进行目录化管理，支持以下分类：
   - **施工日志** - 每日施工记录
   - **检验批** - 工序质量验收记录
   - **材料报验** - 进场材料报验申请
   - **隐蔽工程** - 隐蔽工程验收记录
   - **试验报告** - 材料/工艺试验报告
   - **竣工图** - 竣工图纸归档
   - 各分类支持上传对应资料文件，记录上传人、上传时间、备注
3. **资料上传与查看** - 支持多文件上传，按分类目录展示，可在线预览 PDF / 图片格式，其他格式提供下载链接

## 注意事项
- 施工任务的进度（`actualProgress`）为 0~100 整数，前端进度条展示，后端不做自动计算，由现场管理人员手动更新
- 资料分类建议用字典表维护（`DocumentCategory`），新增分类无需改代码
- 隐蔽工程和检验批与实际施工进度关联，建议在任务详情页展示关联资料列表
- 施工日志建议按日期自动生成模板，减少现场人员填写工作量
- 文件上传同样使用文件存储抽象层，保证存储可切换
